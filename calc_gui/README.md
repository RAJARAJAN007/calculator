# Simple GUI Calculator

A **two-step calculator** built using **Python Tkinter**.  
This calculator allows users to:

1. Choose an operation (Add, Subtract, Multiply, Divide).  
2. Enter two numbers and calculate the result.  
3. Optionally choose a different operation to calculate again.  

---

## Features

- **Two-step workflow:** First choose operation, then enter numbers.  
- **Basic arithmetic operations:** Addition, Subtraction, Multiplication, Division.  
- **Error handling:**  
  - Prevents division by zero.  
  - Validates numeric input.  
- **Multiple calculations:** Users can change numbers or select a new operation.  
- **User-friendly GUI:** Easy to use with clear labels and buttons.  

---

## Requirements

- Python 3.x  
- Tkinter (comes pre-installed with Python)  

---
